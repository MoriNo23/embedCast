# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-31

### Added

- Android TV application for video casting
- WebSocket-based remote control protocol (port 8080)
- Linux CLI tool for host control
- Web GUI for browser-based control
- Resume playback functionality (saves/restores positions)
- Auto-reconnect feature for WebSocket connections
- Animated splash screen with logo
- Leanback-optimized TV interface
- GitHub Actions CI/CD pipeline
- Automatic releases on tags

### Features

- Video casting from Linux host to Android TV
- Play, pause, stop, seek controls
- Quality selection
- Token refresh/reload
- Real-time status synchronization
